# Support

For questions, use GitHub Issues.

## Ask for help

Open an Issue with the **Question / Adoption Help** template.

Include:

- what you are trying to build
- which tools you use
- where the workflow becomes messy
- what kind of handoff or review problem you want to solve

Keep examples general and sanitized.

## Suggest an improvement

Open an Issue with the **Improvement Proposal** template, or submit a Pull Request.

## Professional help

If you want help adapting Sara Agentic OS to your own workflow, open an adoption/help Issue with enough general context for the maintainer to understand the problem.
